Virtual Orchestra System

发表于 2006-02-07 | 更新于 2018-06-23 | 分类于 随笔杂谈

Virtual Orchestra System也就是vos了，呵呵，故弄玄虚一把。

没有什么音乐天赋，所以我起初对这类音乐游戏不感兴趣，一直没玩。大四的时候，空闲时间比较多，所以找了很多可玩的东西。一天心血来潮安了vos，对着电脑屏幕和键盘，想冒充一下音乐人，无奈就连敲键盘的速度都还不算专家级，到底是露怯。这时候，同宿舍的姐妹们正好回来了，我们发起了六手联弹运动（比四手联弹还厉害）来演奏一些级别比较高的vos曲子。Joy玩vos已经两年了，自然负责了主旋律部分的弹奏，而我和椰子则一左一右负责伴奏：）这场弹奏十分壮观，原来单机游戏也可以如此热闹，从此我就喜欢上了vos，并且发现它对疏解压力、郁闷有奇效，每每压力比较大或者很郁闷的时候，就打开它，自顾自地陶醉在所弹奏的不成曲的声音中，真可谓何以解忧，惟有vos。

我比较喜欢钢琴演奏的曲子，很好听，记得以前看过说，莫扎特的曲子为什么那么好听、那么打动人呢？原因是他的曲子契合人脑的某种波长（具体记不清了，大概是某种东西）。我想我喜欢钢琴的声音也是这个理儿吧，尤其是对像我这种对画面的敏感度远超过声音的人来说。玩vos时，最喜欢弹克莱德曼的一系列钢琴曲，不管能弹成什么样儿： P 还有土耳其进行曲和周杰伦的蜗牛等等。比较杂的喜好的说，不过每首弹完都会给我畅快淋漓的舒服感觉，这大概就是我喜欢它们的原因……

前一阵子，看到人家玩劲乐团，感觉它跟vos很像阿，于是想起了六手联弹，想起了那么多弹过的曲子……
